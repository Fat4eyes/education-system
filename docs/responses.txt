401 Unauthorized

    Токен отсутствует или является не валидным.
    Необходимо авторизовать пользователя (получить токен и отправлять его с каждым запросом).

    Пользователю следует показывать сообщение:
        > Вы не авторизованы. Для совершения данного действия вам необходимо авторизоваться.

403 Forbidden

    Токен присутствует и является валидным, но пользователь не содержит роли, которой разрешено выполнять данное действие.

    Пользователю следует показывать сообщение:
        > У вас недостаточно прав для совершения данного действия. Обратитесь к администратору.

500 Internal Server Error

    Внутренняя ошибка сервера.
    
    Возвращается сообщение:
        > Внутренняя ошибка сервера. Попробуйте повторить операцию еще раз.
        > Если ошибка будет повторяться, пожалуйста, обратитесь к администратору.

200 OK

    Ответ без ошибки и без данных:
    {
        "Success": true
    }

    Ответ без ошибки и с данными:
    {
        "Data": {
            "Id": 5934593,
            "Name": "Виктор"
        },
        "Success": true
    }

    Ответ с ошибкой:
    {
        "Error": "Неверная электронная почта или пароль.",
        "Success": false
    }
