401 Unauthorized

    Токен отсутствует или токен является не валидным.
    Что делать? Авторизовать пользователя (получить токен и отправлять его с каждым запросом).

403 Forbidden

    Токен присутствует и является валидным, но пользователь не содержит роли, которой разрешено выполнять данное действие.

500 Internal Server Error

    Внутренняя ошибка сервера.
    Данные никакие не возвращаются.
    Пользователю следует показывать сообщение:
        > Внутренняя ошибка сервера. Попробуйте повторить операцию еще раз.
        > Если ошибка будет повторяться, пожалуйста, обратитесь к администратору.
    Это сообщение будет автоматически возвращаться с сервера.

200 OK

    Ответ без ошибки и без данных:
    {
        "Data": null,
        "Sucess": true
    }

    Ответ без ошибки и с данными:
    {
        "Data": {
            "Id": 5934593,
            "Name": "Виктор"
        },
        "Sucess": true
    }

    Ответ с ошибкой:
    {
        "Error": "Неверный логи или пароль",
        "Sucess": false
    }
